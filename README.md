# Study Flow – Student Productivity Web App

A single-page productivity application that helps students manage tasks,
track study sessions using a Pomodoro timer, and monitor workload
through an analytics dashboard.

## Features
- Task CRUD management
- Pomodoro timer
- Study analytics dashboard
- Persistent task storage using localStorage

## Tech Stack
HTML • CSS • JavaScript
